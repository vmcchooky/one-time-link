# Frontend Scaffold

This app is the browser client for `one-time-link`.

## Planned responsibilities

- create secret form
- client-side encryption with Web Crypto
- reveal gate UI to resist preview bots
- secret decryption in the browser

## Local development

```bash
npm install
npm run dev
```

Set `VITE_API_BASE_URL` if your backend does not run on `http://localhost:8080`.
